# Product Overview

**设计百宝箱 (Design Treasure Box)** is a curated design resource aggregation platform for designers and developers seeking high-quality design aesthetic references.

## Core Features

- Resource browsing by category (8 categories: color tools, CSS templates, fonts, icons, design inspiration, website examples, UI kits, mockups)
- Professional rating system (5 dimensions: overall, usability, aesthetics, update frequency, free level)
- Search and filtering capabilities
- Personal favorites management (localStorage-based)
- Resource recommendations with featured content
- Detailed resource pages with curator notes
- Responsive design (mobile, tablet, desktop)
- Dark/light theme support

## User Experience

- macOS-style dock sidebar navigation
- Masonry grid layout for resource cards
- AI prompt input (bottom floating)
- Smooth animations and transitions
- Accessibility-compliant (WCAG 2.1 AA)

## Data

- 32 curated design resources across 8 categories
- 13 featured resources (rating ≥ 4.0)
- Static JSON data files (categories.json, resources.json)
- Schema validation with Zod
